**0x05-pointers_arrays_strings**
- `0-reset_to_98.c` a function that updates the value it points to to 98
- `1-swap.c` a function that swaps the values of two integers
- `2-strlen.c` a function that returns the length of a string
- `3-puts.c` a function that prints a string, followed by a new line, to stdout
- `4-print_rev.c` a function that prints a string, in reverse, followed by a new line
- `5-rev_string.c` a function that reverses a string
- `6-puts2.c` a function that prints every other character of a string
- `7-puts_half.c` a function that prints half of a string
- `8-print_array.c` a function that prints n elements of an array of integers
- `9-strcpy.c` a function that copies the string pointed to by src, to the buffer pointed to by dest
